# Python Log Monitoring & Disk Space Checker

## 📖 Project Description
This project is a simple Python-based monitoring script used to:
- Analyze application logs for ERROR messages
- Check system disk space
- Exit the program if disk space is low or errors are detected

This type of script is commonly used in **DevOps, Linux administration, and AWS monitoring**.

---

## 🛠 Features
- Counts ERROR occurrences in application logs
- Checks free disk space in GB
- Prints alerts for low disk space
- Uses exit codes for automation pipelines
